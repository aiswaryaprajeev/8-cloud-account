# Cloud Computing  
-	Cloud computing is a technology that allows you to store and access data and applications over the internet instead of using your computerâ€™s hard drive or a local server.
- The cloud provider manages servers, storage, and scalability while users pay only or what they use. 
- It has a front end for user access through browsers or applications and back end where servers handles various functions. 
- Cloud computing improves flexibility, reduces infrastructure costs, and enables access from anywhere with an internet connection.

##Cloud Services
1. Microsoft Azure
-	Free tier/trial includes : 65+ always-free services within defines usage limits (such as storage, AI and Analytics).
-	Duration : 12 months of free services and limited always-free services.
-	Key Limitations : Requires credit card for signup, usage limits apply, some services are region-specific.
2. Amazon Web Services (AWS)
-	Free tier/trial includes : 12-month Free Tier (EC2,S3,RDS).
-	Duration : 12 months for the 12-month tier
-	Key Limitations : Must stay within service limits, no rollover, charges apply if exceeded.
3. Google Cloud Platform (GCP)
-	Free tier/trial : Compute Engine, Cloud Storage, BigQuery, and other services.
-	Duration : 90 days
-	Key Limitations : Credit card required, some services restricted to specific regions, credits expire after 90 days.
